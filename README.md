# Calculateur d&#39;IMC (Indice de Masse Corporelle)

## Description du Projet

Ce projet est une application web simple développée avec Streamlit qui permet de calculer
l&#39;Indice de Masse Corporelle (IMC) d&#39;une personne. L&#39;IMC est une mesure qui évalue la
corpulence d&#39;une personne en fonction de sa taille et de son poids, et indique si elle se situe
dans une catégorie de poids normale, en insuffisance pondérale, en surpoids ou en obésité.

## Fonctionnalités

L&#39;application demande à l&#39;utilisateur de saisir :
* Son poids (en kg)
* Sa taille (en mètres)

Elle calcule ensuite l&#39;IMC et affiche la catégorie de poids correspondante.

## Comment Utiliser l&#39;Application

1. Saisissez votre poids en kilogrammes dans le champ &quot;Poids (kg)&quot;.
2. Saisissez votre taille en mètres dans le champ &quot;Taille (m)&quot;.

3. L&#39;application affichera automatiquement votre IMC et votre catégorie de poids.

## Technologies Utilisées

* [Python](https://www.python.org/)
* [Streamlit](https://streamlit.io/)
